L’écriture des nombres en lettres

Pour écrire tous les nombres jusqu’à 999 999 999 999 en lettres, il existe 26 mots différents:

Un ; deux ; trois ; quatre ; cinq ; six ; sept ; huit ; neuf ; dix ; onze ; douze ; treize ; quatorze ; quinze ; seize ; vingt ; trente ; quarante ; cinquante ; soixante ; cent ; mille ; million ; milliard ; etc.

Ils sont presque tous invariables sauf pour les cas suivants :

Million et milliard s’accordent au pluriel :

Ex : un million ; cinq cent millions

- un milliard ; sept milliards

Cent prend un S quand le nombre termine par cent et qu’il a plusieurs centaines :

Ex : deux mille cinq cent trois ; quatre mille cent ; neuf mille deux cents

Vingt prend un s quand le nombre se termine par quatre-vingts :

Ex : trois cent vingt ; huit cent quatre-vingt-trois ; cent quatre-vingts

Pour t’aider : écris le nombre en chiffres au brouillon.

On met un s à cent seulement quand le nombre se termine par :

200 ; 300 ; 400 ; 500 ; 600 ; 700 ; 800 et 900

On met un s à vingt seulement quand le nombre se termine par 80.

Règle pour les traits d'union : 2 possibilités

- Depuis 1990, on préconise l'usage des traits d'union entre chaque mot du nombre

Ex : soixante-treize-mille-vingt-et-un

- Avant 1990 on met des tirets entre les mots qui forment un nombre plus petit que cent.

Ex : trente-cinq mille six cent cinquante-deux

Il n’y a pas de tiret autour du mot etc.

Ex : soixante-treize mille vingt et un
