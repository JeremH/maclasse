le verbe

Le verbe est le noyau de la phrase. Le verbe peut exprimer soit une action, soit un état :

Ex : Les enfants courent dans le pré.

courent est le verbe (conjugué). Courir est un verbe d'action.

Ex : Je parais grand.

parais est le verbe (conjugué). Paraître est un verbe d'état.

Il est important de bien savoir différencier :

- les verbes d'action (sauter, dormir, courir, venir, aller, prendre, etc...)

- les verbes d'état (être, paraître, sembler, demeurer, rester, devenir, avoir l’air, etc...).

Astuce : Un verbe d'état peut toujours se remplacer par le verbe être. Ex : Je parais grand. Je suis grand.

Le verbe conjugué permet donc d'exprimer dans une phrase une action (ou un état) qui se situe :

- dans le passé : Hier, j'ai fait mes devoirs.

- dans le présent : Aujourd'hui, Je fais mes devoirs.

- dans l'avenir : Demain, je ferai mes devoirs.

Comment reconnaître un verbe dans une phrase ?

Une des astuces, pour repérer facilement un verbe dans une phrase est de changer le temps de cette phrase.

Ex : le chat mange la souris. (présent)

Phrase au passé : Hier, le chat a mangé la souris.

Phrase au futur : Demain, le chat mangera la souris.
